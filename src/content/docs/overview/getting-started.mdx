Elixir UI is a modern, minimal React component library that helps you build beautiful interfaces fast.

---

## 1. Install Dependencies

Make sure Tailwind CSS and Framer Motion are installed:

~~~bash
npm install tailwindcss framer-motion
~~~

---

## 2. Add a Component

Use the CLI to add the Button component:

~~~bash
npx shadcn@latest add https://ui.elixirlabs.in/registry/button.json
~~~

Or copy-paste the Button code from the registry directly.

---

## 3. Use the Component

Import and use the Button:

~~~jsx
import { Button } from './components/ui/button'

export default function Demo() {
  return <Button className="px-4 py-2 bg-blue-600 text-white rounded">Click Me</Button>
}
~~~

---

Elixir UI components are unstyled by default, so customize them easily with Tailwind or your favorite CSS method.
