To get started with Elixir UI, follow these simple steps to set up the required dependencies and add components to your project.

---

## 1. Install Dependencies

Elixir UI relies on Tailwind CSS for styling and Framer Motion for animations. Install them using npm or yarn:

```bash
npm install tailwindcss framer-motion
```
