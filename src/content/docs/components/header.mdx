## Variants

### Sticky Header

Stays fixed at the top when scrolling.

<CodeBlock code={`<Header
  variant="sticky"
  blur={true}
  logo="Brand"
  navLinks={[...]}
/>`} />

### Floating Header

Rounded, centered header that appears on scroll.

<CodeBlock code={`<Header
  variant="floating"
  rounded="full"
  shadow="xl"
  blur={true}
  logo="Brand"
  navLinks={[...]}
/>`} />

### Transparent Header

Perfect for hero sections, becomes solid on scroll.

<CodeBlock code={`<Header
  variant="transparent"
  colorOnScroll={true}
  blur={true}
  logo="Brand"
  navLinks={[...]}
/>`} />

## NavLink Interface

```tsx
interface NavLink {
  label: string;
  href: string;
  badge?: string;        // Optional badge (e.g., "New", "Pro")
  submenu?: NavLink[];   // Optional dropdown submenu
}
```
